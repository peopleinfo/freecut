---
title: Keyboard Shortcuts
description: Complete list of keyboard shortcuts for FreeCut.
---

## Playback

| Action         | Shortcut |
| -------------- | -------- |
| Play / Pause   | `Space`  |
| Previous frame | `Left`   |
| Next frame     | `Right`  |
| Go to start    | `Home`   |
| Go to end      | `End`    |

## Editing

| Action            | Shortcut      |
| ----------------- | ------------- |
| Split at playhead | `Alt+C`       |
| Join clips        | `J`           |
| Delete selected   | `Delete`      |
| Ripple delete     | `Ctrl+Delete` |
| Freeze frame      | `Shift+F`     |

## Clipboard

| Action | Shortcut |
| ------ | -------- |
| Undo   | `Ctrl+Z` |
| Redo   | `Ctrl+Y` |
| Copy   | `Ctrl+C` |
| Cut    | `Ctrl+X` |
| Paste  | `Ctrl+V` |

## Tools

| Action            | Shortcut |
| ----------------- | -------- |
| Selection tool    | `V`      |
| Razor tool        | `C`      |
| Rate stretch tool | `R`      |
| Toggle snap       | `S`      |
| Zoom to fit       | `Z`      |

## Markers & Keyframes

| Action                 | Shortcut  |
| ---------------------- | --------- |
| Add marker             | `M`       |
| Remove marker          | `Shift+M` |
| Add keyframe           | `K`       |
| Toggle keyframe editor | `Ctrl+K`  |

## Source Monitor

| Action              | Shortcut  |
| ------------------- | --------- |
| Mark In (playhead)  | `I`       |
| Mark Out (playhead) | `O`       |
| Mark In (skimmer)   | `Shift+I` |
| Mark Out (skimmer)  | `Shift+O` |
| Insert edit         | `,`       |
| Overwrite edit      | `.`       |

## Groups & Project

| Action         | Shortcut       |
| -------------- | -------------- |
| Group tracks   | `Ctrl+G`       |
| Ungroup tracks | `Ctrl+Shift+G` |
| Save           | `Ctrl+S`       |
| Export         | `Ctrl+E`       |
